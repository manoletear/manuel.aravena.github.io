            <h2 class="text-2xl font-bold text-sky-700 mb-4">La Escalada Imparable: Viviendas vs. Salarios</h2>

            <p class="mb-6 text-slate-600 leading-relaxed">

                El incremento en el costo de la vivienda no ha sido uniforme, pero sí consistentemente alto. Las casas han visto un alza del 62.5% y los departamentos un 55.5% en la última década. En contraste, aunque los salarios medianos reales han crecido (aprox. 53% entre 2011-2023), este avance no ha sido suficiente para compensar la inflación inmobiliaria, creando una brecha de asequibilidad cada vez mayor.

            </p>

            <div class="chart-container">

                <canvas id="preciosVsSalariosChart"></canvas>

            </div>

            <p class="mt-6 text-slate-600 leading-relaxed">

                Esta diferencia sostenida en las tasas de crecimiento significa que, año tras año, se necesitan más salarios para comprar la misma propiedad, un desafío que se profundiza con las condiciones del mercado crediticio.

            </p>

                <h2 class="text-2xl font-bold text-sky-700 mb-4">Tasas Hipotecarias: El Doble Filo</h2>

                <p class="mb-6 text-slate-600 leading-relaxed">

                    Acceder a un crédito hipotecario es hoy significativamente más costoso. Las tasas de interés promedio, que rondaban el 2% antes de 2019, se han más que duplicado, alcanzando niveles entre 4.38% y 5.0% en la actualidad. Este encarecimiento del crédito impacta directamente en el monto del dividendo mensual.

                </p>

                <p class="mb-6 text-slate-600 leading-relaxed">

Esto contrasta fuertemente con el salario promedio nacional, que se sitúa en torno a los $600.000 - $650.000.

            <h2 class="text-2xl font-bold text-sky-700 mb-4">El Pie Inicial y las Ayudas Estatales</h2>

            <p class="mb-6 text-slate-600 leading-relaxed">

                Además de los ingresos, el pie (pago inicial) es un desafío considerable, usualmente entre el 10% y 20% del valor de la propiedad. Programas como FOGAES (Fondo de Garantía Estatal) buscan aliviar esta carga permitiendo financiar hasta el 90% del valor. Sin embargo, si bien FOGAES ayuda con el pie, no resuelve el problema de fondo de los altos requisitos de ingreso mensual para calificar al crédito.

            </p>

            <div class="grid grid-cols-1 md:grid-cols-2 gap-6 items-center">

                    <ul class="list-disc list-inside space-y-1">

                        <li>Normalmente se requiere entre <strong>10% y 20%</strong> del valor total.</li>

                        <li>Para una vivienda de UF 2.000 (aprox. $74 millones), esto implica un ahorro de $7.4 a $14.8 millones.</li>

                    </ul>

                    <h3 class="font-semibold text-lg text-sky-600 mt-4 mb-2">FOGAES: Un Alivio Parcial</h3>

                    <ul class="list-disc list-inside space-y-1">

                        <li>El Estado actúa como aval, permitiendo financiar hasta el <strong>90%</strong>.</li>

                        <li>Reduce la barrera del ahorro inicial.</li>

                        <li><strong>No modifica</strong> los requisitos de renta líquida mensual para el crédito.</li>

        function wrapChartLabels(label, maxWidth = 16) {

            const words = String(label).split(' ');

            let lines = [];

            let currentLine = '';

            for (const word of words) {

                if ((currentLine + word).length > maxWidth && currentLine.length > 0) {

                    lines.push(currentLine.trim());

                    currentLine = '';

                }

                currentLine += word + ' ';

            }

            if (currentLine.trim().length > 0) {

                lines.push(currentLine.trim());

            }

            return lines.length > 0 ? lines : [label];

        }
